Windows下是如何搭建自己的个人博客：

一、环境准备

  • 一个Github账号
  • Hugo安装文件
  • Git

二、Hogo的安装和配置

hugo的下载：

  1. 在https://github.com/gohugoio/hugo/releases地址下找到Windows压缩文件下载
  2. 百度云盘 提取码：y279

1：下载好以后在环境变量中加入hugo所在的位置,如图：

2：如何看hugo安装成没成功，打开cmd，输入hugo version，如图所示，则代表安装成功

3：安装成功后，用Hugo来创建自己的博客，命令如下图所示：hugo new site myblog 注：myblog为文件夹名字

4：我们进入到myblog里面，可以看到里面已经生成了一些文件，我们进入到git的面板中

5：下载设置主题，去https://www.gohugo.org/theme/选择自己喜欢的主题进行下载，如图

6：这里我选择air，以他为例子，具体操作如图显示。注：前提一定要下git并且配置好！

7：这样主题就下载好了。我们可以在配置文件 config.toml 中修改主题，在文件中加入theme = “hugo-theme-air” ，这样我们的主题就配好了。

8：启动我们的博客 hugo server

如图：

在浏览器输入 http://localhost:1313/ 我们就可以看到我们的博客启动好了。

我们就可以写文章了，文章格式以Markdown格式书写。可以将写好的文章放到 content目录下面。然后在启动hugo我们就可以看到我们写的文章了。

三、把博客部署的远端github上

  1. 首先创建仓库，如图

  2. 仓库已经有了，但是个空仓库，接下来敲命令 hugo –baseUrl="地址” –buildDrafts 成功后会在根目录下生成一个public文件

  1. 把这个public传到github仓库上，首先进入到public，如图所示，已经把文件加到了仓库里面

    命令：

    ​ cd public/

    ​ git init

    ​ git add .

    ​ git commit -m "第一次提交"

  1. 把本地public和远程github进行关联

    命令 git remote add arigin https://github.com/qiao-qq/qiao-qq.github.io.git

  2. 然后还有个push命令，git push -u origin master，期间会让你输入账号密码，输入即可成

  3. 我们刷新下github页面，就会看到文件已经传上来了

  4. 最后，我们拿仓库地址访问下，成功，博客已经部署到远端！
